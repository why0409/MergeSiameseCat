# GEMINI.md: 《合成大暹罗》微信小游戏开发蓝图 (Cocos 3.7.4)

## 一、 项目概述 (Project Overview)

* **游戏名称：** 《合成大暹罗》(Merge Siamese Cat)
* **游戏核心：** 2D 物理掉落 + 合成进化（类似“合成大西瓜”）。
* **技术栈：** Cocos Creator 3.7.4 + TypeScript + 微信小游戏 API。
* **当前状态：** 已完成核心玩法闭环（物理掉落、合成逻辑、美术素材替换及死亡判定）。

## 三、 核心技术实现计划 (Technical Roadmap)

### 1. 物理环境重建 (Physics & Scene) ✅
* **Canvas 配置：** 720x1280 (微信小游戏标准竖屏)。
* **2D 物理系统：** 开启物理引擎，设置重力 (Gravity)。
* **边界设定：** 建立左、右、底三面 `Static` 刚体墙壁。
* **预制体设计：** 每级猫咪挂载 `RigidBody2D` (Dynamic) 与 `CircleCollider2D`。

### 2. 核心逻辑 (TypeScript) ✅
* **生成器 (Spawner)：** 监听屏幕点击/拖拽，根据数学比例映射实现精准位置预览与掉落。
* **碰撞合成 (Merge Logic)：** 
    * 监听 `onBeginContact`。
    * 采用 UUID 排序与延迟销毁技术，解决物理步进期间操作节点的报错问题。
    * 在中心点生成 `Level + 1` 节点。
* **坐标转换：** 采用屏幕比例映射方案，避开了相机深度带来的偏移问题。

### 3. 游戏机制 ✅
* **死亡判定：** 顶部红线警戒区域检测，停留超过 2s 触发 GameOver。
* **分数系统：** 实时得分显示与逻辑实现。
* **美术表现：** 引入真实暹罗猫照片，采用 `Mask` 组件实现圆形头像效果。
