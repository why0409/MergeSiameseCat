## 一、 项目概述 (Project Overview)

* **游戏名称：** 《合成大暹罗》(Merge Siamese Cat)
* **游戏核心：** 2D 物理掉落 + 合成进化（类似“合成大西瓜”）。
* **技术栈：** Cocos Creator 3.7.4 + TypeScript + 微信小游戏 API。
* **当前状态：** 已完成核心玩法闭环及稳定性优化，准备发布。

## 三、 核心技术实现计划 (Technical Roadmap) ✅

### 1. 物理环境重建 (Physics & Scene)
* **Canvas 配置：** 720x1280 (适配 3.7.4 标准 UI 渲染)。
* **2D 物理系统：** 开启物理引擎，设置重力 (Gravity: -960)。
* **边界设定：** 建立左、右、底三面 Static 刚体墙壁。
* **预制体：** 1-10 级猫咪预制体完整构建，挂载 `RigidBody2D` 与 `CircleCollider2D`。

### 2. 核心逻辑 (TypeScript)
* **生成器 (Spawner)：** 实现精准的鼠标/触摸位置映射，增加 `isValid` 节点保护防止拖拽冲突崩溃。
* **碰撞合成 (Merge Logic)：** 
    * 采用 UUID 排序确保单次触发。
    * 实施“预览状态无敌”机制，防止顶部猫咪被非法碰撞。
    * 采用“隐藏+延迟销毁”方案，解决物理步进期间操作节点的报错问题。
* **死亡判定 (Deadline)：** 采用 Map 记录区域内物体，增加无效节点自动清理逻辑，停留 2s 触发 GameOver。

### 3. 美术与 UI
* **素材应用：** 10 级真实暹罗猫照片完整导入。
* **视觉美化：** 采用 `Mask (ELLIPSE)` 组件实现完美的圆形头像效果。
* **UI 系统：** 实时得分显示，最高分记录逻辑就绪。
